# AI News Aggregator

AI-powered news aggregation system that collects, processes, ranks, and summarizes AI-related news from multiple sources.

## Features

- Scrapes news from OpenAI, Anthropic, and YouTube sources
- AI-generated summaries and daily digests
- Content ranking and filtering pipeline
- PostgreSQL database integration
- Automated email delivery system
- Modular backend architecture

## Tech Stack

- Python
- PostgreSQL
- OpenAI API
- Web scraping pipelines
- Email automation

## Run Locally

```bash
git clone https://github.com/antilume/AI-NEWS-AGGREGATOR.git
cd AI-NEWS-AGGREGATOR
pip install -r requirements.txt
python main.py
